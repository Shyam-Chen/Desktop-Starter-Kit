## Getting Started

1. Clone this Boilerplate

```bash
$ git clone --depth 1 https://github.com/Shyam-Chen/Electron-Chromium-Starter.git <PROJECT_NAME>
$ cd <PROJECT_NAME>
```

2. Install Dependencies

```bash
$ yarn install
```

3. Run the Application

```bash
$ yarn start
```

Reload: Cmd + R <br>
Close: Ctrl + C

4. Build the Application

```bash
$ yarn build
```
